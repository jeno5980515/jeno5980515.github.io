[Java] Shallow Copy和Deep Copy

前言

在複製陣列的時候，要注意是需要Shallow Copy還是Deep Copy，直接看程式碼。

Shallow Copy

可以看到當a改變的時候b也跟著改變了，這是因為b是reference到a陣列，但如果要個別獨立就必須使用Deep Copy。

Deep Copy

簡單來說就是先配置記憶體位置給b，再將a的內容逐項複製過去。
