[Machine Learning] Supervised和Unsupervised Learning

前言

最近在看Coursea的Machine Learning，做個筆記以免忘記。

Supervised Learning(監督式學習)

在Training data中必須標注類型，好讓機器判斷。

例如這是一個腫瘤良性與惡性的資料，藍色代表良性而紅色代表惡性，因此預測的粉色歸類在良性的機率比較大。

Unsupervised Learning(非監督式學習)

在Training data沒有標注類型，機器自動分堆。

雞尾酒派對問題，將聲音分別抓出來。
